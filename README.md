# StellarPayoutLabs Frontend

Frontend application for the Stellar payout platform.

## Scope

- Operator dashboard for payout operations.
- Approval and review flows for high-risk payouts.
- Status views for payout batches and transactions.
- Wallet/network configuration for Stellar environments.

## Tech Stack

- React + TypeScript + Vite
- Minimal dependency footprint
- API-driven architecture (talks to `backend` service)

## Development

```bash
npm install
npm run dev
```
